KentekenAPI

Open Source RESTful JSON-LD API met Open Data van het RDW

Download Download View KentekenAPI on GitHub

Over deze API

Deze API wordt ontwikkeld ten behoeven van Case 5 in samenwerking met Case 4 van PiLOD 2.0. Doel van deze API is het onderzoeken van JSON-LD (http://json-ld.org) in combinatie met een standaard RESTful JSON webservice bovenop de XML webservice die RDW aanbiedt.

API Documentatie

De documentatie van de KentekenAPI staat op Apiary.io. Door een link met Github wordt elke wijziging aan apiary.apib in de KentekenAPI repository automatisch gesynced naar Apiary waardoor de documentatie altijd up-to-date is en voorzien is van versiebeheer. Tevens biedt Apiary een mock mogelijkheid om de API te testen.

  • Documentatie: http://docs.kentekenapi.apiary.io
  • Mock-server: http://mock-kentekenapi.apiary.io

Contribute

Om mee te ontwikkelen aan deze API kun je simpelweg de repository forken en een pull request doen. Dit geldt voor zowel de ontwikkeling van de code als het nader te bepalen JSON-LD design, welke via Markdown in het apiary.apib kan worden toegepast.
